# phishing simulator

a simulator to know how phishing attack it's work.

- use python 3
- use HTML  and CSS
- log the works

## Installation

Install the dependencies and devDependencies and start the server.

```sh
git clone https://github.com/MahyarMasoomi/phish_simulator.git
cd phish_simulator
chmod +x app.py
```

start the server

```sh
./app.py
```

- the server is run in your local host at http://localhost:8080
- the dasboard is in the http://localhost:8080/dashboard

## see the IP User Password and the other logs

run this code in yor bash after the target login

```sh
python3 analys_log.py
```
